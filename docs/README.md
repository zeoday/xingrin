## 快速导航

### 常见问题
- **版本不一致**：参考 [版本管理 - 故障排查](./version-management.md#故障排查)
- **镜像拉取失败**：检查网络和 Docker Hub 连接
- **远程 Worker 部署**：参考版本管理文档中的部署流程

### 核心概念
- **IMAGE_TAG**：版本锁定机制，确保所有组件版本一致
- **--pull=missing**：镜像拉取策略，优先使用本地缓存
- **task_distributor**：负载感知任务分发器
